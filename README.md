# Requirement

Ensure the following steps to finish the deployment:

- you run this project on `PHP 8.x`
- you have correct values in those config files. (esp. `$STATICCS_CONFIG`)
- you have proper cross-site settings
- you have installed the `Fileinfo` extension
- you have set pseudo-static rules according to `pseudo-static.txt`
- you have adjust your PHP & Nginx/OpenResty config to allow file uploading (limit file size as well)
